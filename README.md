# dwellur site

A minimal Next.js landing page for dwellur.

## How to use

1. Upload/import this folder into a GitHub repository.
2. Import the repo into Vercel.
3. Add your generated hero image as:

```txt
public/dwellur-hero.jpg
```

If that image is missing, the page shows a simple built-in phone mockup fallback.

## Edit contact email

The contact email is currently:

```txt
support@dwellur.com
```

## Run locally

```bash
npm install
npm run dev
```
